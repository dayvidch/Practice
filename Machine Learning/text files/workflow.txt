3. Model validation
    mean absolute error
    prediction error = actual - predicted value
    with MAE, take absoulte value of each error and find the average. "On average, my predictions are off by"

    from sklearn.metrics import Mean_absolute_error
    predicted = model.predict(X)
    mean_absoulte_error(y, predicted)

    train/test/split

    from sklearn.model_selection import train_test_split
    train_X, val_X, train_y, val_y = train_test_split(X, y, random_state=0)
    model.fit(train_X, train_y)

    predictions = model.predict(val_X)
    mean_absolute_error(val_y, predictions)
